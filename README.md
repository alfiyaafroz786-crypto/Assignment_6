# Simple Node Server

## Routes (http://localhost:4000)

1) 'home' or '/'
This route show home page where we have logo, navbar and username in the header section below it main section where images button and it content is there.

2) 'services'
This route show service page where we have heading "Services Page".

3) 'about'
This route show about us page where we have heading "About Us Page".

4) 'contact'
This route show contact us page where we have heading "Contact Us Page".

4) '/anyotherroute'
This route show 404 Not Found because we create only home, about, contact and services routes.
