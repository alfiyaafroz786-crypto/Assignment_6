# Simple Node Server

## Routes (http://localhost:4000)

1) 'home' or '/'
This route show home page where we have heading, navbar and some contents.

2) 'services'
This route show service page where we have heading "Services Page".

3) 'about'
This route show about us page where where we have heading, navbar and some contents.

4) 'contact'
This route show contact us page  where we have heading, navbar and email address to contact.

4) '/anyotherroute'
This route show 404 Not Found because we create only home, about, contact and services routes.
